Pulse Recognition



The web application is entirely built on Python.The application involves the usage of Python's micro framework Flask. The web app can be used for identifying the pulse shown in the image. The classifier can classify the image into 6 categories - Arhar,Urad,Toor,Moong,Chana and Masoor Dal.
